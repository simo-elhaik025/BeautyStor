# BeautyStor - Vision du projet

## Objectif

Développer une plateforme e-commerce professionnelle, évolutive et maintenable, dédiée dans un premier temps à la vente de produits cosmétiques au Maroc.

L'architecture devra permettre l'ajout futur d'autres catégories de produits (vêtements, accessoires, etc.) sans refonte majeure.

## Technologies prévues

- Backend : Spring Boot (API REST)
- Base de données : MySQL
- Frontend : React
- Gestion des versions : Git & GitHub
- Documentation : Markdown

## Objectifs pédagogiques

- Maîtriser Spring Boot en pratique.
- Concevoir une API REST professionnelle.
- Concevoir une base de données relationnelle.
- Appliquer les bonnes pratiques Git/GitHub.
- Documenter un projet logiciel.
- Suivre une démarche de gestion de projet.
